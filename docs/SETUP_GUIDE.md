# Setup Guide

## Step 1: Prerequisites
- Databricks account
- Python 3.9+

## Step 2: Access Databricks
1. Go to your Databricks workspace
2. Create a cluster

## Step 3: Run Notebooks
Run in order:
1. `01_bronze_ingestion.py` - Load data
2. `02_silver_cleaning.py` - Clean data
3. `03_gold_features.py` - Engineer features
4. `04_clustering_analysis.py` - Cluster
5. `05_visualization_insights.py` - Visualize

## Step 4: View Results
Check Databricks dashboards
