
# Customer Segmentation Analysis

Segment e-commerce customers using K-Means clustering for targeted marketing strategies.

## Objective
Identify distinct customer segments based on behavioral and demographic data to enable:
- Targeted marketing campaigns
- Personalized recommendations
- Customer retention strategies
- Revenue optimization

## Dataset
- **Records**: 50,000 customers
- **Features**: 26 behavioral & demographic attributes
- **Source**: Ecommerce customer churn dataset
